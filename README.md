# PawFinder

PawFinder is an Android app to help users find lost and adoptable pets, view nearby shelters, and manage pet profiles.

## Features
- Browse adoptable pets with photos and details
- Search for nearby shelters and pet services
- Save favorite pets and shelters
- Simple pet profile management for adopters

## Requirements
- JDK 11 or newer
- Android Studio or Gradle with Android SDK installed
- Gradle wrapper (included)

## Build
From the project root (Windows):

```powershell
.\gradlew.bat assembleDebug
```

Or (Unix):

```bash
./gradlew assembleDebug
```

## Run
- Open the project in Android Studio and run the `app` module on an emulator or device.
- Or install the debug APK from the command line:

```powershell
.\gradlew.bat installDebug
```

## Project structure
- `app/` — Android application module
- `gradle/`, `build.gradle.kts`, `settings.gradle.kts` — build configuration
